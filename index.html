Details & Summary

What is Dashboard.css?

A classless CSS framework for dashboards. No class names, no IDs — style via semantic HTML and data-variant attributes.

How do variants work?

Add data-variant="success|danger|warning|info|accent" to <mark>, <button>, <blockquote>, or <small>.

Is it responsive?

Yes. Sidebar collapses on mobile, nav shows a hamburger, cards reflow via auto-fill grid, and the table scrolls horizontally inside its wrapper.
